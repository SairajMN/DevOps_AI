# Changelog

All notable changes to the DevOps AI VS Code Extension will be documented in this file.

## [1.0.0] - 2026-02-22

### Added

- Initial release
- Chat interface with DevOps AI assistant
- Log analysis functionality
- Code fix suggestions
- Run tests command
- Build project command
- Install dependencies command
- Analyze current file command
- Quick chat command
- Status check command
- Start server command
- Keyboard shortcuts for common actions
- Configuration options for server URL, default model, auto-analyze, and inline diagnostics
- Context menu integration for Python, JavaScript, and TypeScript files
- Activity bar icon and sidebar view

### Features

- Natural language commands for DevOps tasks
- Integration with DevOps AI server
- Support for multiple AI models (DeepSeek R1, DeepSeek V3, Llama 70B, Mixtral, Qwen, Gemini Flash)
- Inline diagnostics for detected errors
- Auto-analysis option for log files
